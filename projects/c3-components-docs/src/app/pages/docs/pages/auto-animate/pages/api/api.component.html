Auto Animate API

Import

import { C3AutoAnimateDirective } from 'c3-components';

Selector

[c3AutoAnimate]

Inputs

Name Type Default Description
options AutoAnimateOptions | AutoAnimationPlugin undefined Configuration options for the animation behavior.

AutoAnimateOptions

Property Type Default Description
duration number 250 Animation duration in milliseconds.
easing string 'ease-in-out' CSS easing function for the animation.
disrespectUserMotionPreference boolean false Whether to ignore user's reduced motion preference.

Usage Notes

  • Apply the directive to a parent element that contains dynamic children.
  • Works with @for, *ngFor, and @if structural directives.
  • Animations are automatically triggered when children are added, removed, or reordered.
  • Respects prefers-reduced-motion media query by default.
