PDF Viewer API

Import

import { PdfViewerContainerComponent } from 'c3-components';

Components

PdfViewerContainerComponent

Selector: pdf-viewer-container

Inputs

Name Type Description
src string URL or path to the PDF file to display. Required.

Signals

Name Type Description
pdfViewer Signal<PDFViewer | null> Reference to the PDF.js viewer instance.
pdfDocument Signal<PDFDocumentProxy | null> Reference to the loaded PDF document.

Sub-components

PdfViewerHeaderComponent

Selector: pdf-viewer-header

Provides navigation controls (page navigation, zoom).

PdfViewerContentComponent

Selector: pdf-viewer-content

Renders the actual PDF pages.

Dependencies

This component requires pdfjs-dist to be installed:

npm install pdfjs-dist
