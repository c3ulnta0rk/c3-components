PDF Viewer

A standalone PDF viewer component built on top of PDF.js, providing a complete viewing experience with navigation controls, zoom, and page controls.

Basic Usage

Requires PDF.js worker files to be available in your assets folder. See the API section for setup instructions.

Setup Instructions

1. Copy PDF.js worker files to your assets:

// angular.json assets array
"assets": [
  {
    "glob": "**/*",
    "input": "node_modules/pdfjs-dist/build/",
    "output": "/pdfjs/"
  }
]

2. Configure the worker path in your app:

import * as pdfjs from 'pdfjs-dist';
pdfjs.GlobalWorkerOptions.workerSrc = '/pdfjs/pdf.worker.min.mjs';
