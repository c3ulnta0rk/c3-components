Modal API

Status

Coming Soon

This component is currently in development. The module C3ModalModule is available but does not yet export any components.

Planned Features

  • Lightweight modal component
  • Backdrop and close button support
  • Animation options
  • Configurable size and positioning

Alternative

In the meantime, consider using:

  • C3DialogService for dialog-based modals with full functionality
  • Angular Material MatDialogModule directly
// Using C3DialogService
import { C3DialogService } from 'c3-components';

constructor(private dialog: C3DialogService) {}

openModal() {
  this.dialog.createDialogFromComponent({
    component: MyModalComponent,
    width: '500px'
  });
}
