Trace Card API

Import

import {
  C3TraceCardModule,
  C3TraceCardComponent,
  C3TraceCardContentComponent,
  C3TraceCardTitleComponent,
  C3TraceCardAlignLeftDirective,
  C3TraceCardAlignRightDirective
} from 'c3-components';

C3TraceCardComponent

Selector: c3-trace-card

Host Class: c3-trace-card

A card component designed for displaying trace/timeline-like content with left or right alignment.

Signals

Name Type Default Description
align Signal<'left' | 'right'> 'left' Alignment of the card. Controlled via directives.

Host Bindings

Class Condition
.align-left When align() === 'left'
.align-right When align() === 'right'

Content Projection

Component Description
c3-trace-card-title Title section of the card.
c3-trace-card-content Main content section of the card.

C3TraceCardTitleComponent

Selector: c3-trace-card-title

Title component for the trace card. Typically displays a heading or label.

C3TraceCardContentComponent

Selector: c3-trace-card-content

Content component for the trace card. Contains the main body of the card.

Alignment Directives

C3TraceCardAlignLeftDirective

Selector: c3-trace-card[align-left]

Sets the card alignment to left.

C3TraceCardAlignRightDirective

Selector: c3-trace-card[align-right]

Sets the card alignment to right.

Usage Example

<!-- Left-aligned card (default) -->
<c3-trace-card>
  <c3-trace-card-title>
    Event Title
  </c3-trace-card-title>
  <c3-trace-card-content>
    <p>Event description and details...</p>
  </c3-trace-card-content>
</c3-trace-card>

<!-- Right-aligned card -->
<c3-trace-card align-right>
  <c3-trace-card-title>
    Another Event
  </c3-trace-card-title>
  <c3-trace-card-content>
    <p>This card is aligned to the right.</p>
  </c3-trace-card-content>
</c3-trace-card>

<!-- Explicit left alignment -->
<c3-trace-card align-left>
  <c3-trace-card-title>
    Third Event
  </c3-trace-card-title>
  <c3-trace-card-content>
    <p>Explicitly aligned left.</p>
  </c3-trace-card-content>
</c3-trace-card>

Usage Notes

  • The alignment directives are standalone and can be imported individually.
  • The align signal is automatically set by the directives in the constructor.
  • Use CSS to style the timeline connector line between cards.
  • Both align-left and align-right are attribute directives.
