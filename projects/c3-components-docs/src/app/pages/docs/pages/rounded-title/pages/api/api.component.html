Rounded Title API

Import

import { C3RoundedTitleComponent } from 'c3-components';
// or
import { C3RoundedTitleModule } from 'c3-components';

C3RoundedTitleComponent

Selector: c3-rounded-title

A pill-shaped title container that wraps projected content with a rounded background. Styled via the c3-rounded-title-theme SCSS mixin.

Content Projection

The component uses <ng-content> to project any content inside a .title wrapper element.

Host Bindings

Property Value Description
class c3-rounded-title CSS class applied to the host element for theming.

Styling

The component uses ViewEncapsulation.None so it can be styled globally via the theme mixin:

@use 'c3-components' as c3;

@include c3.c3-rounded-title-theme($your-theme);

The mixin applies primary color to the .title inner element and accent color to the outer pill background.

Usage Example

<c3-rounded-title>Section Title</c3-rounded-title>

<c3-rounded-title>
  <mat-icon>settings</mat-icon> Settings
</c3-rounded-title>
