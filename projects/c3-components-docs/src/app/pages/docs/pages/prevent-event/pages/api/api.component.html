Prevent Event API

Import

import {
  C3PreventClickDirective,
  C3StopPropagationDirective,
  C3PreventKeyboardEventDirective
} from 'c3-components';

C3PreventClickDirective

Selector: [c3-prevent-click]

Prevents the default action and stops event propagation on click events. Useful for links or buttons that should not trigger their default behavior.

Behavior

Event Actions
click preventDefault() + stopPropagation()

C3StopPropagationDirective

Selector: [c3-stop-click-propagation]

Only stops immediate propagation of click events, without preventing the default action. Allows the element to perform its default action while not bubbling to parents.

Behavior

Event Actions
click stopImmediatePropagation()

C3PreventKeyboardEventDirective

Selector: [c3-prevent-keyboard]

Prevents default action and stops propagation for keyboard events.

Behavior

Event Actions
keydown preventDefault() + stopPropagation()

Use Cases

  • Preventing navigation on anchor elements used as buttons
  • Stopping click events from bubbling to parent containers
  • Creating overlay close buttons that don't trigger underlying elements
  • Preventing keyboard shortcuts from triggering default browser behavior
