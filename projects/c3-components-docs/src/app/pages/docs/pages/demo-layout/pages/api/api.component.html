Demo Layout API

Overview

The Demo Layout is a documentation-specific layout component used for showcasing component examples. It is not part of the exported c3-components library.

Purpose

This layout is used internally within the c3-components documentation site to provide a consistent structure for component demonstrations and examples.

Features

  • Consistent spacing and typography for examples
  • Code snippet display
  • Interactive component previews
  • Responsive layout for documentation pages

Note

If you're looking to create similar documentation layouts in your own project, consider using:

  • Standard Angular layout components
  • CSS Grid or Flexbox for responsive layouts
  • Code highlighting libraries like highlight.js or prism.js
