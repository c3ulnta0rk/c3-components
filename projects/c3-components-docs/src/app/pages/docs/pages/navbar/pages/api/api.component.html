Navbar API

Import

import { C3NavbarComponent } from 'c3-components';
// or
import { C3NavbarModule } from 'c3-components';

C3NavbarComponent

Selector: c3-navbar

A simple navigation bar component with Material elevation. Content is projected via <ng-content>.

Host Bindings

Property Value Description
class c3-navbar mat-elevation-z4 CSS classes applied to the host element for theming and elevation.

CSS Classes

Class Description
.fixed Add to the host to make the navbar fixed-positioned with offset for the side menu.
.flex Add to the host to make link items evenly distributed with flexbox.
.selected Add to an <a> child to mark it as the active/selected link.

Styling

The component uses ViewEncapsulation.None. Apply the theme mixin globally:

@use 'c3-components' as c3;

@include c3.c3-navbar-theme($your-theme);

Usage Example

<c3-navbar>
  <a routerLink="/home" routerLinkActive="selected">Home</a>
  <a routerLink="/settings" routerLinkActive="selected">Settings</a>
</c3-navbar>
