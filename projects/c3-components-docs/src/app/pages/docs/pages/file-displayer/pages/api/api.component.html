File Displayer API

Import

import {
  C3FileDisplayerModule,
  C3FileDisplayListComponent,
  C3FileDisplayGridComponent,
  C3FileDisplayCardComponent,
  C3FileDisplayIconComponent,
  C3FileDisplayListColumnDefDirective,
  C3FileDisplayListHeaderCellDefDirective,
  C3FileDisplayListCellDefDirective
} from 'c3-components';

C3FileDisplayListComponent

Selector: c3-file-display-list

Displays files in a list/table format with customizable columns.

Inputs

Name Type Default Description
files Array<FileMetadata & { location: string }> required Array of files to display.
config C3FileViewerConfig undefined Configuration for the file viewer.
displayFn (file: FileMetadata & { location: string }) => string (file) => file.name Function to generate display name for files.
columns string[] ['icon', 'name', 'date'] Column names to display.
deletable boolean false Whether files can be deleted.

Outputs

Name Type Description
onDelete FileMetadata Emits when a file is deleted.

Methods

Method Parameters Description
openDialog(event, index) MouseEvent, number Opens the file viewer dialog for the specified file.
getColumnDef(columnName) string Returns the column definition for the specified column name.

C3FileDisplayGridComponent

Selector: c3-file-display-grid

Displays files in a grid of cards.

Inputs

Name Type Default Description
files Array<FileMetadata & { location: string }> required Array of files to display.
config C3FileViewerConfig undefined Configuration for the file viewer.
cardSize C3FileCardSize '32' Size of the card thumbnails.
displayFn (file: FileMetadata & { location: string }) => string (file) => file.name Function to generate display name for files.
deletable boolean false Whether files can be deleted.

Outputs

Name Type Description
onDelete FileMetadata Emits when a file is deleted.

C3FileDisplayCardComponent

Selector: c3-file-display-card

Individual file card with thumbnail preview.

Inputs

Name Type Default Description
fileObjectUrl (FileMetadata & { objectUrl?: Observable<string> }) | null required The file metadata with optional object URL for preview.
displayFn (file: FileMetadata) => string (file) => file.name Function to generate display name.
deletable boolean false Whether the delete button is shown.
size C3FileCardSize '32' Size of the card.

Outputs

Name Type Description
onDelete void Emits when the delete button is clicked.
onDownload void Emits when the download button is clicked.

Computed Properties

Property Type Description
getFileType 'image' | 'application' | 'video' | 'unknown' Returns the file type category.
getFileObjectUrl Observable<string> | null Returns the object URL observable for the file.
getFileName string Returns the computed file name.

C3FileDisplayIconComponent

Selector: c3-file-display-icon

Displays an appropriate icon based on file MIME type.

Inputs

Name Type Default Description
fileType string required MIME type of the file (e.g., 'image/png', 'application/pdf').

Icon Mapping

MIME Type Icon Color Class
image/* image text-primary
video/* play_circle text-secondary
application/pdf picture_as_pdf text-error
application/msword description text-success
Other content_copy text-gray

Column Definition Directives

Use these directives to define custom columns in c3-file-display-list.

C3FileDisplayListColumnDefDirective

Selector: [c3FileDisplayListColumnDef]

Input Type Description
c3FileDisplayListColumnDef string The column title/name (required).
classList string[] CSS classes to apply to the column.

C3FileDisplayListHeaderCellDefDirective

Selector: [c3FileDisplayListHeaderCellDef]

Template for the column header cell.

C3FileDisplayListCellDefDirective

Selector: [c3FileDisplayListCellDef]

Template for each row cell in the column.

C3FileCardSize

Type definition for card sizes:

type C3FileCardSize = '16' | '32' | '64' | '128' | '256';

Usage Example

<!-- Grid Display -->
<c3-file-display-grid
  [files]="files"
  [cardSize]="'64'"
  [deletable]="true"
  (onDelete)="handleDelete($event)"
></c3-file-display-grid>

<!-- List Display with Custom Columns -->
<c3-file-display-list
  [files]="files"
  [columns]="['icon', 'name', 'size', 'date']"
>
  <ng-container c3FileDisplayListColumnDef="size">
    <ng-template c3FileDisplayListHeaderCellDef>Size</ng-template>
    <ng-template c3FileDisplayListCellDef let-file>
      {{ file.size | number }} bytes
    </ng-template>
  </ng-container>
</c3-file-display-list>
