Expansion API

Import

import {
  C3ExpansionModule,
  C3ExpansionComponent,
  C3ExpansionHeaderComponent
} from 'c3-components';

C3ExpansionComponent

Selector: c3-expansion

Host Class: c3-expansion

An expandable panel component with smooth animations powered by C3AutoAnimateDirective.

Inputs

Name Type Default Description
headerClass string 'c3-expansion-header' CSS class for the header section.
isExpanded boolean false Controls the expanded state. Two-way bindable with [(isExpanded)].
options AutoAnimateOptions | AutoAnimationPlugin undefined Animation options passed to the auto-animate directive.

Methods

Method Description
toggleExpand() Toggles the expanded state.

Content Projection

Element Description
c3-expansion-header The clickable header section.
Default content The expandable body content.

C3ExpansionHeaderComponent

Selector: c3-expansion-header

Host Class: c3-expansion-header

Header component that triggers expansion on click.

Inputs

Name Type Default Description
title string undefined Optional title text for the header.
isExpanded boolean false Current expanded state (for styling purposes).

Outputs

Name Type Description
toggleExpand void Emits when the header is clicked to toggle expansion.

Usage Example

<c3-expansion [(isExpanded)]="panelOpen">
  <c3-expansion-header [title]="'Click to expand'">
    <!-- Optional custom header content -->
    <mat-icon>expand_more</mat-icon>
  </c3-expansion-header>

  <!-- Expandable content -->
  <div class="panel-content">
    <p>This content is shown when expanded.</p>
  </div>
</c3-expansion>

Styling

Import styles from:

@use 'c3-components/lib/c3-expansion/styles/c3-expansion';

Usage Notes

  • The expansion animation is powered by @formkit/auto-animate.
  • The header automatically subscribes to toggle the expansion state.
  • Use [(isExpanded)] for two-way binding of the expanded state.
  • The component uses C3AutoAnimateDirective as a host directive.
  • Content is conditionally rendered based on the expanded state.
