Dropdown API

Import

import {
  C3DropdownModule,
  C3DropdownComponent,
  C3DropdownTrigger
} from 'c3-components';

C3DropdownComponent

Selector: c3-dropdown

Container for dropdown menu content. Must be used with C3DropdownTrigger.

Inputs

Name Type Default Description
panelClass string | string[] | Set<string> | { [key: string]: any } undefined CSS classes to apply to the dropdown panel. Supports ngClass syntax.

Signals

Name Type Description
isOpen Signal<boolean> Whether the dropdown is currently open.
template Signal<TemplateRef<unknown>> Reference to the dropdown content template.

C3DropdownTrigger

Selector: [c3DropdownTrigger]

Export As: c3DropdownTrigger

Directive that triggers the display of a dropdown menu on click.

Inputs

Name Alias Type Default Description
dropdown c3DropdownTrigger C3DropdownComponent required Reference to the dropdown component. Uses model.required().
dropdownDisabled - boolean false Whether the trigger is disabled.
dropdownClass - string | string[] | Set<string> | { [key: string]: any } '' CSS classes for the dropdown menu.

Methods

Method Description
show() Programmatically open the dropdown.
close() Programmatically close the dropdown.

Overlay Positioning

The dropdown uses Angular CDK Overlay with the following positions:

Position Behavior
Primary Below the trigger, aligned to start
Fallback Above the trigger if no space below

Usage Example

<button [c3DropdownTrigger]="myDropdown">
  Open Menu
</button>

<c3-dropdown #myDropdown [panelClass]="'my-custom-panel'">
  <div class="dropdown-item">Option 1</div>
  <div class="dropdown-item">Option 2</div>
  <div class="dropdown-item">Option 3</div>
</c3-dropdown>

Styling

The dropdown panel has the class c3-dropdown-panel by default.

Import styles from:

@use 'c3-components/lib/c3-dropdown/styles/c3-dropdown';

Usage Notes

  • The dropdown automatically closes when clicking outside or on the backdrop.
  • Uses a transparent backdrop that captures clicks.
  • Implements NG_VALUE_ACCESSOR for form integration.
  • The trigger listens to click events by default.
