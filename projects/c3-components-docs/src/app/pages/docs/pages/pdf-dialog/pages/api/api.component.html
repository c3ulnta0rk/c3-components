PDF Dialog API

Import

import { PdfViewerContainerComponent } from 'c3-components';

Note: This component is currently exported from the internal path and may change in future versions.

Overview

The PDF Dialog functionality uses the PdfViewerContainerComponent in combination with C3DialogService to display PDF files in a dialog window.

PdfViewerContainerComponent

Selector: pdf-viewer-container

A container component for rendering PDF files using PDF.js.

Inputs

Name Type Description
src string URL or path to the PDF file. Required.

Signals

Name Type Description
pdfViewer Signal<PDFViewer | null> Reference to the PDF.js viewer instance.
pdfDocument Signal<PDFDocumentProxy | null> Reference to the loaded PDF document.

Usage in Dialog

import { C3DialogService } from 'c3-components';

// Open PDF in a dialog
openPdfDialog(pdfUrl: string) {
  this.dialog.createDialogFromComponent({
    component: PdfViewerContainerComponent,
    data: { src: pdfUrl },
    width: '80vw',
    height: '90vh',
    panelClass: 'pdf-dialog'
  });
}

Sub-components

PdfViewerHeaderComponent

Selector: pdf-viewer-header

Provides navigation controls (page navigation, zoom).

PdfViewerContentComponent

Selector: pdf-viewer-content

Renders the actual PDF pages.

Dependencies

This component requires pdfjs-dist to be installed:

npm install pdfjs-dist

Configure the worker file path in your application.
