Flowing Menu API

Import

import {
  C3FlowingMenuModule,
  C3FlowingMenuComponent,
  C3FlowingMenuItemComponent,
  C3FlowingMenuItemContentComponent
} from 'c3-components';

C3FlowingMenuComponent

Selector: c3-flowing-menu

Host Class: c3-flowing-menu

A tabbed menu component that displays content from the selected menu item with smooth transitions.

Inputs

Name Type Default Description
selectedIndex number 0 Index of the currently selected menu item. Two-way bindable with [(selectedIndex)].

Methods

Method Parameters Description
selectMenu(index) number Programmatically select a menu item by index.

Content Children

Query Type Description
c3FlowingMenu QueryList<C3FlowingMenuItemComponent> Collection of menu item children.

C3FlowingMenuItemComponent

Selector: c3-flowing-menu-item

Individual menu item containing both the menu label and associated content.

Signals

Name Type Description
active Signal<boolean> Whether this menu item is currently active.
menuTpl Signal<TemplateRef | undefined> Reference to the menu label template.
contentTpl Signal<TemplateRef | undefined> Reference to the content template.

Content Projection

Use ng-template with the following markers:

Template Ref Description
#menuTpl Template for the menu label/tab.
#contentTpl Template for the content displayed when this item is selected.

C3FlowingMenuItemContentComponent

Selector: c3-flowing-menu-item-content

Wrapper component for menu item content. Used inside c3-flowing-menu-item.

Usage Example

<c3-flowing-menu [(selectedIndex)]="activeTab">
  <c3-flowing-menu-item>
    <ng-template #menuTpl>
      <span>Tab 1</span>
    </ng-template>
    <ng-template #contentTpl>
      <c3-flowing-menu-item-content>
        <p>Content for Tab 1</p>
      </c3-flowing-menu-item-content>
    </ng-template>
  </c3-flowing-menu-item>

  <c3-flowing-menu-item>
    <ng-template #menuTpl>
      <span>Tab 2</span>
    </ng-template>
    <ng-template #contentTpl>
      <c3-flowing-menu-item-content>
        <p>Content for Tab 2</p>
      </c3-flowing-menu-item-content>
    </ng-template>
  </c3-flowing-menu-item>
</c3-flowing-menu>

Styling

Import styles from:

@use 'c3-components/lib/c3-flowing-menu/styles/c3-flowing-menu';

Usage Notes

  • The component uses ngSkipHydration for SSR compatibility.
  • Content is dynamically rendered using ViewContainerRef.createEmbeddedView().
  • Menu items are queried using contentChildren().
  • Use [(selectedIndex)] for two-way binding of the active tab.
  • The selected content is displayed in a designated content area.
