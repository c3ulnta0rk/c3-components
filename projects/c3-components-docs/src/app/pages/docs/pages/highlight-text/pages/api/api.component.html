Highlight Text API

Import

import { C3HighlightTextDirective } from 'c3-components';
// or
import { C3HighlightTextModule } from 'c3-components';

C3HighlightTextDirective

Selector: [c3HighlightText]

A directive that highlights matching text within the host element's content. Wraps matched text in <span class="c3-highlight"> elements.

Inputs

Name Type Default Description
c3HighlightText string '' The text string to search for and highlight.
c3HighlightTextCaseSensitive boolean false Whether the matching should be case-sensitive.

CSS Classes

Class Description
.c3-highlight Applied to the <span> wrapping matched text. Style this class to customize the highlight appearance.
.c3-highlight_container Applied to the wrapper <span> that contains the highlighted text node.

Usage Example

<!-- Basic usage (case-insensitive by default) -->
<p [c3HighlightText]="searchTerm">Some text to search in</p>

<!-- Case-sensitive matching -->
<p [c3HighlightText]="searchTerm"
   [c3HighlightTextCaseSensitive]="true">
  Case Sensitive Text
</p>

Usage Notes

  • The directive operates on the DOM directly, walking text nodes within the host element.
  • When the search text changes, previous highlights are automatically reset before new ones are applied.
  • Special regex characters in the search text are escaped automatically.
  • The directive is standalone and can be imported directly without a module.
