Highlight Text API

Status

Coming Soon

This component is currently in development. The module C3HighlightTextModule is available but does not yet export any components.

Planned Features

  • Text highlighting pipe or directive
  • Search term highlighting
  • Customizable highlight styles
  • Case-insensitive matching
  • Multiple term support

Planned Usage

<!-- As a pipe -->
<p [innerHTML]="text | c3HighlightText: searchTerm"></p>

<!-- As a directive -->
<p [c3HighlightText]="searchTerm">{{ text }}</p>
