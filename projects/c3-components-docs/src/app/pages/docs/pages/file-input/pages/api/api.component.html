File Input API

Import

import { C3InputFileModule, C3DownloadService } from 'c3-components';

C3InputFileComponent

Selector: c3-input-file

Inputs

Name Type Default Description
accept string required File types to accept (e.g., 'image/*', '.pdf').
multiple boolean false Whether to allow multiple file selection.
baseUrl string 'localhost:3000' Base URL for the upload endpoint.
dest string 'api/upload' Destination path for uploads.
method string 'POST' HTTP method for the upload request.
headers Object undefined Custom headers to include in the upload request.

Outputs

Name Type Description
fileAdded EventEmitter<C3InputFile> Emitted when a file is added and upload starts.

Methods

Name Description
click() Programmatically open the file picker dialog.

C3InputFile Class

Represents an uploaded file with progress tracking.

Properties

Name Type Description
fileName string Original file name.
mimetype string MIME type of the file.
size number File size in bytes.
progression number Upload progress (0-100).
progressionChange Subject<number> Observable for progress changes.
response any Server response after upload completes.

Methods

Name Description
emit('abord') Cancel the ongoing upload.

C3DownloadService

Service for downloading files as Blobs.

Methods

Name Parameters Returns Description
download url: string Observable<Blob> Downloads a file from the given URL.
